# Snsproject
### django sns project

</br>

### 프로젝트 목적

1. Django MVT 패턴을 익힌다.

    - Model, View, Templates

2. Django ORM(Object Relational Mapping)에 대해 배운다.

3. Django Abstrectuser를 사용해 로그인과 회원가입을 손쉽고 간단히 구현한다.

4. 게시글 읽기, 쓰기, 삭제 기능.

5. Many-To-Many로 팔로우 기능.

6. django-taggit 기능.

</br>
